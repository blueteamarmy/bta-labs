Ataque Reconstruído — Kill Chain Completa

Você analisou múltiplas fontes de log e montou a sequência completa do ataque.

O ataque começou às 02:00 com um scanner automatizado (sqlmap) varrendo o servidor web de 10.20.45.113. Foram tentadas path traversal e SQL Injection para identificar vulnerabilidades.

Paralelamente, às 02:17, brute force SSH contra o host linux01.corp.local resultou em acesso root após 47 tentativas. Um web shell foi carregado em /var/www/html/shell.php para persistência de acesso web.

Com acesso ao domínio, o atacante criou a conta svc_helper no DC01 às 02:19 e elevou seus privilégios ao grupo Administrators. Um novo serviço (WinDefend_Helper) e uma tarefa agendada garantiram persistência mesmo após reinicialização.

A partir de 03:18, beacon C2 para update-cdn185.net a cada 300s indicou controle ativo. A exfiltração ocorreu via DNS tunneling (dados encoded nos subdomínios) e transferência HTTPS de 10,2 MB em 2 minutos.
